Anchor Link Examples

These are valid HTML examples with structural components.

Questions

  1. How do Assistive Technology (AT) User Agents (UA) present them?
  2. If UA does not present the markup attributes, is it a UA or Web content issue?

Examples

Simple
href="#"
Expectation: UA presents link destination
Target new window
href="#" target="_blank"
Expectation: UA presents HREF AND behavior as opening in a new window.
External relationship
a href="#" rel="external"
Expectation: UA presents HREF AND as off-site link.
Help
a href="#"
Tel:
a href="tel:#"
Expectation: UA prompts user to confirm opening link in new application or to place call.
Mailto:
href="mailto:#"
Expectation: UA opens link in system email.
PDF Download
download type="pdf"
Expectation: UA indicates HREF is PDF document for download.
Language
href="#" hreflang="fr"
Expectation: UA indentifies HREF document as French document
